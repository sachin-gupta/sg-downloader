#REFERRED URLs

- [Sample Project - Kennethreitz](https://github.com/kennethreitz/samplemod): An architecture with sphinx document generation and basic setup.py.
 Single one-level package named sample with modules inside.
- [Writing Setup Script](https://docs.python.org/2/distutils/setupscript.html): Technical docs from Python designers, says about things
to be used in the file. Details out all technical aspects of writing Setup.py
- [Hitchhker's Guide to Packaging](https://the-hitchhikers-guide-to-packaging.readthedocs.io/en/latest/quickstart.html):A guide to layout your project, writing setup.py, creating first release and register at PyPI, & uploading.
- [Setup Guide - Kennethreitz](https://github.com/kennethreitz/setup.py): A cool guide with powerful setup.py file
  - For example, this setup.py provides a $ python setup.py upload command, which creates a universal wheel (and sdist) and uploads your
  package to PyPi using Twine, without the need for an annoying setup.cfg file. It also creates/uploads a new git tag, automatically.
